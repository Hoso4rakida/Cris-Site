# Ana Interiores - Website

## Overview
This is a static HTML/CSS/JavaScript website for an interior design company called "Ana Interiores". The site showcases architectural projects and house plans.

## Project Structure
- `index.html` - Main HTML page
- `css/style.css` - Custom styles
- `js/index.js` - JavaScript functionality
- `img/` - Image assets (logos, slides, icons)

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3.6 (via CDN)
- Bootstrap Icons (via CDN)
- Google Fonts (Oswald, Rajdhani)

## Running the Project
The project uses Python's built-in HTTP server to serve static files:
```bash
python -m http.server 5000 --bind 0.0.0.0
```

## Deployment
This is a static website that can be deployed directly. The public directory is the root folder (`.`).
